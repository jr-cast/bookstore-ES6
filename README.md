## 	:toolbox: Getting Started


<!-- Installation -->
### :gear: Installation

Install my-project with npm

```bash
  npm install
```

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/jr-cast/bookstore.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

* To start server please use LiveServer extension
